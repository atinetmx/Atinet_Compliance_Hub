Objetivo del sistema:

Migrar el sistema ATINET (VB 6.0 / VB.NET + MySQL 5.0) a una plataforma web moderna, manteniendo compatibilidad con:

operación local en notarías

operación centralizada para ATINET

escalabilidad futura

monitoreo en tiempo real

Stack base propuesto:

Backend: Laravel

Frontend: Blade + React (Inertia)

DB: MySQL 8+

Sync: APIs REST + Jobs + Queues

Auth: Laravel Sanctum

Infraestructura: On-Premise + Central (Híbrido)
